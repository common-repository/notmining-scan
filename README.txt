Check if your website was infected with cryptojacking.

== Description ==
Con este plugin puedes comprobar si tu web está infectada mediante cryptojacking, malware, si está en alguna lista negra, etc. Además sabrás el script exacto que está infectado y podrás limpiar el código sin necesidad de buscar a mano en toda la web.

Características:

1. Puedes ver cual es el script exacto que está infectado.
2. Sabrás si alguna vez hemos detectado que tu web hubiese estado infectada.
3. Tendrás tu web monitorizada frente a malware.

https://www.youtube.com/watch?v=eLBrTXakA0M

Este plugin es un servicio de NotMining. Puedes ver nuestros terminos y condiciones y nuestra política de privacidad.

[Términos y condiciones](http://notmining.org/terminos.html "Términos y condiciones")
[Política de privacidad](http://notmining.org/privacy.html "Política de privacidad")

== Installation ==
1. Descarga NotMining Scan desde el repositorio de Wordpress y súbelo a \"wp-content/plugins\" o descárgalo e instálalo directamente desde tu Worpress
2. Activa el plugin.
3. Registra tu dominio en http://plugin.notmining.org/.
4. Recarga la página del plugin
